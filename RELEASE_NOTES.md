# MailcowClient v0.0.10 — First Release

> ⚠️ **Unofficial Release** — This is an early, community-driven release of MailcowClient. It is **not** an official release from the Mailcow/mailcow-dockerized project. Use at your own risk.

## What is MailcowClient?

MailcowClient is an open-source, cross-platform desktop email client built specifically for [Mailcow](https://mailcow.email/) self-hosted mail servers. It provides a native desktop experience for managing email, calendars, and contacts on your own Mailcow instance.

Download the appropriate file for your platform from the **Assets** section below.

## Features in this release

- 📧 **Email** — Read, send, and organise email via IMAP/SMTP
- 📅 **Calendar** — Full CalDAV calendar support with event creation and editing
- 👤 **Contacts** — CardDAV contacts management
- 🔐 **Multi-account** — Connect multiple Mailcow accounts simultaneously
- 🎨 **Modern UI** — Clean, dark-themed interface with violet accent colours

## Known limitations

- macOS builds are for Apple Silicon (arm64) only; Intel (x64) builds are not yet provided
- Code signing is not yet set up; your OS may show an "untrusted developer" warning — you can safely dismiss it
- This is an early release; bugs and missing features are expected
